# DuplicateFileRemover
Python automation project to detect and delete duplicate files using MD5 checksum.

## Features
- Calculates MD5 checksum of files
- Detects duplicate files
- Deletes duplicate files automatically
- Works recursively on directories

## Technologies Used
- Python
- hashlib
- os

## How to Run

```bash
python 151_CheckSumCalculateDirectoryStoreFinal.py
```
